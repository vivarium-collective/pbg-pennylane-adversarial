Architecture

The PennyLaneAdversarialProcess wraps a QML classifier built with PennyLane's StronglyEntanglingLayers template and TorchLayer integration. The pipeline progresses through:

  • Training — supervised cross-entropy optimization on the PlusMinus dataset (Adam optimizer).
  • Benign evaluation — accuracy on clean test samples.
  • PGD Attack — projected gradient descent perturbation (epsilon-bound L_inf, sign-gradient ascent).
  • Adversarial evaluation — accuracy under PGD attack.
  • Adversarial retraining — training with augmented dataset containing original + perturbed samples.
  • Robust evaluation — accuracy on perturbed data after retraining.

Inputs arrive through PennyLane Datasets; outputs emit accuracy, loss, and query-count metrics through process-bigraph ports. Each update() advances one epoch or evaluation phase.
